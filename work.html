As a web coder and designer, my task for this midterm was to create a single-page website based on the given prototype design. I carefully followed the layout specified in the prototype, while also incorporating the provided content and images. The header section featured an image with the site title positioned on top of it using the appropriate HTML tags.

In the article section,I ensured that the text wrapped around the image correctly. When displaying the facts, I utilized proper syntax to list them in an organized manner. For the form, I linked all the input fields to their corresponding labels, ensuring a seamless user experience. Throughout the development process, I strived to replicate the mockup design as closely as possible, while allowing for changes in subject, images, and color as needed.
