I designed a bakery website called Jenny's Bakery and applied my coding skills.

I built a 4-page website. I made my website fully functional and visually appealing. I made the site respond to different screen sizes.I made use of CSS grid, flexbox to create the layout of the page. I made media queries and specific breakpoints. I followed accessibility guidelines using proper contrast, alt text for images, skip links, aria roles, proper HTML syntax and schema information.

I made all the images on the site should be optimized for web use, and responsive image techniques were incorporated. All images had a minimum of 2 image sizes for optimizing images for small and large screens. I also made two noticeable animations that was used to enhance the overall user experience of the site.
